# 🚢 Titanic Survival Prediction

> Predict whether a passenger would survive the Titanic disaster — powered by XGBoost and deployed as an interactive web app.

## 🎯 Overview
Built an end-to-end machine learning pipeline that takes passenger details as input and predicts survival probability in real time.

## ✨ Features
- ~84% accuracy using XGBoost classifier
- Feature engineering: Title extraction, FamilySize, IsAlone flags
- Handles missing data and categorical encoding
- Interactive web interface built with Streamlit

## 🛠️ Tech Stack
| Category | Tools |
|----------|-------|
| Language | Python |
| ML Model | XGBoost, Scikit-learn |
| Frontend | HTML, CSS, JavaScript |
| Deployment | Streamlit |
| Data | Pandas, NumPy |

## 📊 Model Performance
| Metric | Score |
|--------|-------|
| Accuracy | ~84% |
| Features Used | Pclass, Sex, Age, Fare, Embarked, Title, FamilySize, IsAlone |
